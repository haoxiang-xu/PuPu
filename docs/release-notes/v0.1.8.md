# PuPu 0.1.8 — Early Preview Release 🚀

A desktop AI client with a file-system chat explorer, memory, workspace context, and runtime management for both local and cloud models.

## What's New in 0.1.8

- 🎨 **Make PuPu yours — full theme customization** — a new theme editor lets you pick your own colors with a live preview, start from built-in presets, and import/export or reset themes. Colors now flow through a semantic palette, so the sidebar, surfaces, cards, and modals all stay consistent in both light and dark.
- 📦 **Tools that just install — local runtime auto-setup** — MCP servers that need **Node (npx)** or **UV (uvx)** now get their runtime resolved and set up automatically. No more installs that silently fail because the environment wasn't ready.
- ⚠️ **Clear install errors** — when a toolkit install does fail, you now see the actual reason right on the toolkit card and detail panel, instead of a silent dead end.

### Fixes & Polish

- 🧭 **Smarter scrolling** — landing correction and scroll navigation for the message window, plus a bottom-viewport inset so the last message and the minimap aren't clipped.
- 🔗 **Trace chain finality** — fixed trace-chain completion ownership so tool/reasoning frames close out reliably (#155, #66).
- 🎨 **Semantic theming under the hood** — shell, input, context menu, card, modal, and markdown think-block backgrounds are all routed through semantic tokens for cleaner light/dark parity.

## ⚠️ Known Limitations

This is still an **early preview**. Some features are incomplete and edge-case bugs may still exist. If you hit one, please open an issue.

---

## On the Horizon

Following the v0.1.5 convention, a short forward-looking note — roadmap previews, not release commitments. Scope and timing may change.

- **App-wide error toasts** — install errors now surface where they happen; the next step is a consistent, app-wide toast so any failure (not just toolkit installs) explains itself clearly and never fails silently.
- **Ultra Apps & Generative UI** — PuPu is exploring constrained, generative surfaces where the assistant can compose and adjust parts of the interface itself. The theme color picker is an early probe of that direction.
