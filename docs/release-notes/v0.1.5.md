# PuPu v0.1.5 Release Notes Draft

## What to expect

Starting with upcoming releases, PuPu release notes will include a short forward-looking section. The goal is to keep users and watchers aware of the project direction while keeping shipped changes and future work clearly separated.

- **Agent Builder** — PuPu is moving toward a visual builder for reusable agents. The direction is to let users define an agent's prompt, model defaults, tool access, and sub-agent pool, then select that agent directly from chat for repeatable workflows.
- **Plan Skills** — PuPu will explore plan-oriented skills for longer work. The intent is to make complex tasks easier to break down, review, and resume by giving agents a more structured way to create and follow project-specific plans.

These are roadmap previews, not release commitments. Scope and timing may change as the features are tested in real workflows.
