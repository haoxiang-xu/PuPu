- Step 3.1
  1. ① Icon + 状态点 + count — Artifacts 入口图标在 chat header 右上角；右上角小点表示「有 draft」（黄）/「全 finalized」（绿）；旁边小数字表示 artifact 总数（>1 时显示）。没 artifact 时 icon 灰掉或隐藏。
  2. ② Type pill + title — 最左 pill 表示 artifact type（plan / text / file...）；title 是 artifact title。多个 artifact 时右侧追加 prev/next 切换器。
  3. ③ Status + revision pill — plan 有 status（draft · r2）；workspace 没 status 就显示 r1。
  4. ④ Close — 关闭 panel。

符合「极简 styling」原则：有 border 就不加 background；pill 用 border + 灰字；status 通过小色点而不是大色块。
