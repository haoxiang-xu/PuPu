给 CEO（Haoxiang Xu / founder）做增长/COO 汇报时，**以原始数据为主体，诚实、直接、全面地展示数字本身**。商业解读、"so what"、行动建议是有价值的加分项，但必须作为清晰分离的一层叠加在数据之上——**绝不能替代、压缩或筛掉底层数据**。

**Why:** CEO 在 2026-06-09 明确反馈：他认可分析做得好，但更希望先看到完整、不被加工的数字，自己形成判断。过多的叙事/自我解读有风险——会遮蔽或带偏原始数据，让他看不到全貌。

**How to apply:**
- **先把完整数据表摆全**：所有拉到的指标都列出来，不要为了"简洁"而省略 metric。该有的就有，宁全勿删。
- **诚实直接**：数据能说明什么、不能说明什么都讲清楚；无 baseline 的增量、403 缺失的 traffic、单快照算不出的指标，都明确标注，不含糊、不粉饰。
- **解读单独成层**：把"so what"/判断/建议放在清楚区隔的板块（如 Recommendations），不要把它揉进数据里、也不要让它替数据发言。
- **不要为了讲故事去裁剪数据**：叙事服从数据，不是数据服从叙事。
