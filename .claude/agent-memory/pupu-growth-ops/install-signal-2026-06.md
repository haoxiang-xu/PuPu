PuPu 安装意图（安装包下载）是最强真实增长信号。基于 2026-06-09 与 2026-06-14 两次快照的可验证 5 天 delta：

- **v0.1.6**（2026-05-25 发布）：06-09 累计 47 → 06-14 累计 64，**5 天 +17（≈3.4/天）**；生命周期速率 ≈3.2 dl/天。
- v0.1.5：73 累计，≈1.49 dl/天；v0.1.4：63 累计，≈1.0 dl/天。两者 5 天 delta 均为 0（老版本已停止吸量，用户在迁移到新版）。

**判定：v0.1.6 是历史安装速率最高的版本，且仍在加速。** 这与 clone 噪声相反——是真信号：新版发布速率明显高于前两版，老版归零说明用户确实在升级而非分散。

**平台偏好：** v0.1.6 三平台几乎均衡（mac arm64 23 / win 22 / linux deb 19），不像早期 mac 独大。说明 Windows/Linux 用户群已成形，发布必须三平台齐发，不能只顾 mac。

**Why 这条重要：** CEO 数据优先且最看重「有没有人真在装」。安装速率上升 = 发布节奏值得加快。
**How to apply:** 每次巡船都 diff 最近 release 的 download_count 算 dl/天，与上一版比；新版速率若低于前一版要 flag 为回归风险。当前结论支持「加快发布节奏」的运营建议。相关：[[clone-spike-2026-06]]、[[silent-downloader-gap]]。
