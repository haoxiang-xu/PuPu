Haoxiang Xu（haoxiang-xu，haoxiangxu1998@gmail.com）是 PuPu 的创始人兼几乎唯一的核心贡献者（247 commits，第二名社区贡献者仅 7）。亲自开了仓库里几乎所有 issue。

**项目阶段（2026-06）：** 早期。stars 33、forks 3、watchers 1。月度发版节奏（v0.1.0→v0.1.6，2 月以来 7 个版本）。下载量级是几十/版本，不是几千。

**他如何看待数据：** 会主动盯 GitHub traffic 图（paths 里 /graphs/traffic 有访问），对 clone 这类表面数字敏感、会主动追问——需要我做反噪声的把关人，而不是附和。

**How to apply:** 在这个量级，绝对数字小是正常的，要按 PuPu 自身趋势打分而非绝对规模。真正的赢是：安装包下载/天上升、出现外部 issue/PR（说明有沉默下载者开始发声）、独立访客上升。真正的忧是：单人 bus factor、issue 几乎全是他自己开（用户在沉默下载、反馈渠道缺失）。给建议要低成本、可单人执行。相关：[[clone-spike-2026-06]]。
